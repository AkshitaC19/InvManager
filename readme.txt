STEPS TO RUN THE CODE

System Requirements:-
I.	MySQL database.
II.	JDK(JDK 8) installed.
III.	Hibernate 

Steps:-
i.	Create database with name "InvManager" in MySQL server.
ii.	Extract InvManager.zip(source code file) and open in IDE(Netbeans).
iii.	Run src/Models/DatabasePopulator.java servlet for database tables initialization.
iv.	Do registration and login to use all functionalities.
